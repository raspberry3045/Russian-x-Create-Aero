# Русский
## Что это?
**Неофициальный** ресурспак, добавляющий перевод мода **[Create Aeronautic](https://modrinth.com/mod/create-aeronautics)** (и некоторых других) на русский язык. Переведено все, что есть в модах.

_Будет пополняться..._
</details>

## Установка
1. Скачайте Create Aeronautics (необязательно, но иначе ничего не произойдет)
2. Скачайте сам ресурспак
3. Поместите архив в папку <code>resoursepacks</code>
4. Включите в самом Майнкрафте
